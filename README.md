### 📌 Featured projects

**[sherlock](https://github.com/liliandoublet/sherlock)** Production-grade French NLP pipeline. Built with uv, Pydantic, Typer CLI, loguru, MLflow, CamemBERT, pytest and GitHub Actions CI.
